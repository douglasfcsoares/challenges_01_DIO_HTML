Tag de título ou cabeçalhos do H1 ao H6.

Voltar

As tags h1 ao h6 são usadas para definir cabeçalhos no HTML. h1 define o cabeçalho mais importante. h6 define o título menos importante. é recomendado que se use apenas um h1 por página ele representa o cabeçalho/assunto principal de toda a página. não é recomendado pular os níveis dos títulos, após usar o h1 use o h2 e assim sucessivamente.

Tag P.

Voltar

A tag p define um parágrafo. Os navegadores adicionam automaticamente uma linha em branco antes e depois de cada p.

Tag Mark.

Voltar

A tag Mark define um texto que deve ser marcado ou destacado.

Tag Small.

Voltar

A tag Small define um texto menor (como direitos autorais e outros comentários laterais).

Tag I.

Voltar

A i tag define uma parte do texto em uma voz ou humor alternativo. O conteúdo interno geralmente é exibido em itálico. A tag i é frequentemente usada para indicar um termo técnico, uma frase de outro idioma, um pensamento, um nome de navio, etc. Use o i elemento somente quando não houver um elemento semântico mais apropriado.

Tag U.

Voltar

A tag u marca representa algum texto não articulado e com estilo diferente do texto normal, como palavras com erros ortográficos ou nomes próprios em texto chinês. O conteúdo interno geralmente é exibido com um sublinhado.

Tag Strong.

Voltar

A tag Strong é usada para definir o texto com forte importância. O conteúdo interno geralmente é exibido em negrito.

Tag Ol.

Voltar

A tag Ol define uma lista ordenada. Uma lista ordenada pode ser numérica ou alfabética.

Tag Ul.

Voltar

A tag Ul define uma lista não ordenada. (marcada geralmente com um ponto.)

Tag A.

Voltar

A tag A define um hiperlink, que é usado para vincular de uma página a outra. O atributo mais importante do A elemento é o href atributo, que indica o destino do link.

Tag Hr.

Voltar

A tag Hr define uma quebra temática em uma página HTML gerando uma linha horizontal na página. (por exemplo, uma mudança de tópico).

Tag Sub.

Voltar

A tag Sub define o texto subscrito. O texto subscrito aparece meio caractere abaixo da linha normal e, às vezes, é renderizado em uma fonte menor. O texto subscrito pode ser usado para fórmulas químicas, como H 2 O.

Tag Sup.

Voltar

A tag Sup define o texto sobrescrito. O texto sobrescrito aparece meio caractere acima da linha normal e, às vezes, é renderizado em uma fonte menor. O texto sobrescrito pode ser usado para notas de rodapé, como WWW [1].

Tag Blockquote.

Voltar

A tag Blockquote tag especifica uma seção citada de outra fonte.

Tag Font. (Descontinuada no HTML 5)

Voltar

A tag Font foi usada no HTML 4 para especificar a face da fonte, o tamanho da fonte e a cor do texto.

Tag Del

Voltar

A tag Del define o texto que foi excluído de um documento. Os navegadores geralmente traçam uma linha através do texto excluído.

Tag Abbr

Voltar

A tag Abbr define uma abreviação ou um acrônimo, como "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM".
